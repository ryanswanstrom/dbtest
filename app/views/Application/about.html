About dbtest

dbtest is speed comparison test between MongoDB hosted at MongoHQ and Xeround's MySQL-like cloud-hosted database. The speed comparison does inserts and queries.

What does dbtest do?

After pressing the Speed Test button, a specified message is entered for a certain number of iterations. The user supplies both the message and the number of iterations. Each entry is automatically assigned a random level from 0 to 999. That process is timed for both MongoHQ and Xeround. Then a query is run for each of the levels from 0 to 999. This results in 1000 queries being run against MongoHQ and another 1000 being run against Xeround. The inserts and queries are timed and the results are displayed on the results page.

Setup

Currently, the MongoDB is hosted with the free plan from MongoHQ. The Xeround account is a beta account. Both databases have an index setup on level.

The code is hosted at GitHub.

Results

It appears that MongoHQ is much faster for inserts (by a factor of 2 or 3). The query speed of MongoHQ is typically about twice as fast as Xeround. I am thinking the speed of MongoHQ would increase with a paid plan.
